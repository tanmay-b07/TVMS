# TVMS Android App

## Overview

TVMS Android App is a companion application for the TrailBox Vehicle Monitoring System (TVMS).

The application provides quick access to the live TVMS dashboard from Android devices, allowing users to monitor sensor data remotely without manually opening a web browser.

---

## Features

* Mobile access to TVMS dashboard
* Real-time monitoring interface
* Lightweight application
* Simple and user-friendly design
* Direct connection to live dashboard

## Purpose

The application improves accessibility by providing a dedicated Android interface for the TVMS monitoring platform.

This release focuses on mobile dashboard access and serves as the foundation for future mobile-specific features.
