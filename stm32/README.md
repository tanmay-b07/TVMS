# STM32 Firmware

This directory contains the STM32F407 firmware source code used in the TVMS project.

## Contents

* Core source files
* STM32 HAL drivers
* CubeMX project configuration (.ioc)

## Function

The STM32 is responsible for:

* Reading sensor values using ADC
* Processing sensor data
* Transmitting data to ESP32 through UART

## Development Environment

* STM32CubeIDE
* STM32CubeMX
* STM32F407

## Version

TVMS V1.0.3
